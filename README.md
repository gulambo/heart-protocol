# HeartProtocol

> Every autonomous machine that shares our world answers to a verified human heart.

HeartProtocol is a decentralized verification layer for bipedal robots — where human heartbeat is the key to safety authorization, governance, and accountability. We build the trust infrastructure that lets humans stay in control of the machines they share space with.

## Architecture Framework
- **programs/heart_protocol**: On-chain Anchor/Rust security tracking engine.
- **circuits**: ZK-SNARK pulse constraint verification framework.
- **app**: Cross-platform native mobile monitoring and override gateway.
